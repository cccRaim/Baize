## Description

本项目使用 [Nest](https://github.com/nestjs/nest) 构建.

原项目为 [微精弘PHP服务端](https://github.com/zjutjh/wejh-server), 因一些原因需要重构，故诞生此项目。

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
